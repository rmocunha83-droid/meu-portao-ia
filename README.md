# Meu Portão IA

MVP navegável para visualizar estilos de portões em fachadas e captar pedidos de orçamento.

Site publicado: https://meu-portao-ia.vercel.app

## Rodar localmente

1. Instale as dependências:
   `npm install`
2. Inicie o projeto:
   `npm run dev`
3. Abra o endereço exibido no terminal.

## Páginas

- `/` - Landing page
- `/simular` - Upload, preferências, geração simulada e formulário de lead
- `/empresas` - Benefícios, planos e cadastro de parceiros
- `/privacidade` - Política de privacidade do MVP

Os dados ficam apenas no estado local e são registrados no console. O código contém os pontos preparados para futuras integrações de geração de imagem, storage, banco de dados e CRM.
